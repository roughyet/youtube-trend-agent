# YouTube Search Trend Analysis Bot

A Telegram bot that analyzes YouTube **search trends** (not views) from Bangladesh and provides AI-powered insights.

## Features

✅ **Search Volume Analysis** - Analyzes MOST SEARCHED topics, not most viewed  
✅ **38 Topics** - Class 1-10, Educational queries, Tech, English learning  
✅ **Top 20 Rankings** - Ranked by search volume (most searched first)  
✅ **AI Insights** - Gemini analyzes trends and provides recommendations  
✅ **24/7 Operation** - Runs in cloud, works even when your device is off  
✅ **Completely Free** - No payment required (free APIs + cloud tier)

## What It Does

1. **Collects** YouTube search volume for 38 topics
2. **Ranks** by search count (highest to lowest)
3. **Analyzes** trends with Google Gemini AI
4. **Sends** formatted report to Telegram
